bash note

2014-04-12
QA
Ubuntu

bash命令

  • 命令与文件补全功能： ([tab] 按键的好处)

使用 [tab] 按键的时机依据 [tab] 接在命令后或参数后而有所不同。

[Tab] 接在一串命令的第一个字的后面，则为命令补全；
[Tab] 接在一串命令的第二个字以后时，则为『文件补齐』！

所以说，如果我想要知道我的环境中，所有可以运行的命令有几个？ 就直接在 bash 的提示字符后面连续按两次 [tab] 按键就能够显示所有的可运行命令了。
那如果想要知道系统当中所有以 c 为开头的命令呢？就按下 c[tab][tab]就好啦！
